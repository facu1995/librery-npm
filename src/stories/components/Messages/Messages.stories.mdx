# < Message /> ⚙️

El componente Message muestra una imagen alusiva al tipo de mensaje que busca comunicar ('success', 'confirm' o 'error'). Opcionalmente recibe dos props más para mostrar texto describiendo el error o la acción a tomar. 
Este componenente no tiene un tamaño propio, está hecho para adaptarse a un contenedor.

<span>Este componenente no tiene un tamaño propio, está hecho para adaptarse a un contenedor.</span>

```
<contenedor>
    <Message type='error' message1='Oops, hubo un error' />
<contenedor>
```
