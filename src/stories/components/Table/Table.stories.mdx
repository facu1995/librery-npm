# < Table /> ⚙️

<br /> 


Este componente nos sirve para mostrar un arreglo en forma de lista, y ofrece la opción de recibir una función a ejecutar cuando se haga click en la selección de algún producto, pasando ese mismo producto como parámetro.

```
<Table listItem={} listHeaderItem={} onClick={} />
```

<span>NOTA:</span> en la story aparece una notificación con el objeto seleccionado. Esta función está pasada a la story para demostrar el uso de la propiedad onClick y no está incluida en la librería
