# < Paginator /> ⚙️


Este componente cumple la función de informar al usuario cuando está viendo contenido paginado y ofrece capacidad para manejar ese paginado.

```
<Paginator handleChange={()=>{}} />

```

## Propiedades currentPage y maxPages


Además del handleChange, Paginator puede recibir dos props más.
<ul>
    <li><span>currentPage : </span> representa la página actual, su valor por defecto es 1. Para un uso correcto, debería actualizarse cada vez que se realiza una solicitud de una página más.</li>
    <li><span>maxPages :</span>representa el número de la última página disponible. El valor de currentPage no puede exceder el valor de maxPages</li>

</ul>
