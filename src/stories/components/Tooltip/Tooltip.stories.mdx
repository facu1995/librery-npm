# < Tooltip /> ⚙️

<br /> 


El componente tooltip se incluye como hijo del elemento en el que se quiere incluir. Se mantiene oculto hasta que el usuario final hace hover sobre su padre (el elemento que lo contiene) y desaparece cuando el mouse deja de estar sobre él. Puede contener ya sea texto o un link.

```
<ParenElement>
    <Tooltip info='Texto a mostrar' />
</ParenElement>
```

## link

Si pasamos un valor a la propiedad link, el tooltip rendeará un elemento anchor en su interior con el valor de la prop 'link' como su 'href'

```
<ParenElement>
    <Tooltip info='Texto con link' link='https://unlink.dominio' />
</ParenElement>
```
