# < Counter /> ⚙️

Counter es un componente pensado para proyectos donde debemos reflejar una cantidad variable de manera sencilla y en tiempo real. Genera un contenedor que muestra de un lado la cantidad, y del otro lado una descripción de la entidad que se está contando. 

```
<Counter amount={16} label='Componentes' />

```
*<Counter amount={16} label='Componentes' />*

<br />

## Enumerar un arreglo en lugar de mostrar un valor


Además de pasar explícitamente el valor a mostrar, podemos pasar un arreglo y el contador va a devolver el largo del mismo. 


```
const array = [1, 2, 3, 4, 5]
<Counter label='Items' items={array} />

```
*<Counter label='Items' items={[1, 2, 3, 4, 5]} />*

<br />

## Usar contador como botón

El contador puede tomar una propiedad onClick, que recibe una función a ejecutar cuando se clickeé el contador. Además cambia su UI.

```
<Counter label='Clickable' amount={5} onClick=(()=>{}) />

```

*<Counter label='Clickable' amount={5} onClick={()=>{}} />*
