# < Nav /> ⚙️

El componente Nav muestra una lista de links creadas a partir de su propiedad routes, que toma un arreglo de objetos siguiendo la interfaz RouteData. Dependiendo de los valores recibidos, el elemento Nav puede ser vertical, horizonal, tener o no sombra y ser o no colapsable con el toque de un botón.

```
NavProps {
  routes: RouteData[];
  shadow?: boolean;
  button?: boolean;
  direction?: 'row' | 'column';
}
RouteData {
    path: string;
    element: React.JSX.Element;
    icon: any;
}
```
### Notas

Este componente usa la librería react-router-dom, por lo que debe estar envuelto en algun componente <a href="https://reactrouter.com/en/main/router-components/browser-router" target="blank">Router</a> para ser usado.
