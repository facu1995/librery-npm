# < Button /> ⚙️

Componente reutilizable para renderear una etiqueta button. A tráves de sus props podemos modificar su contenido y su comportamiento ante la interacción del usuario, así como manejar su disponibilidad

Su única propiedad obligatoria es 'label', que representa el nodo de texto dentro del botón. 

```
<Button label='Demo Button' />

```
*<Button label='Demo Button' />*

<br />

## Tipos de botón 


Un botón puede ser visible pero no estar habilitado para el usuario hasta que se cumpla cierta condición. como sucede en el botón 'Confirmar' de [DynamicForm](https://storybook.infosis.tech/?path=/story/infosis-buildingblocks-forms--adaptable-form), que está desabilitado hasta que se cumplan todas las validaciones. Podemos accionar este comportamiento con la prop 'type', que acepta uno de N valores definidos en la librería (revisar propiedades en las stories). Para un botón que no se puede clickear, usamos type='disabled'.


```
<Button label='Disabled Button' type='disabled'/>

```
*<Button label='Disabled Button' type='disabled'/>*
